# aaFAT
#### Absolutely awful FAT
aaFat is a simple FAT8 like filesystem.
###### Features:
- Under 1000 lines of C
- No calloc or malloc
- Very simple

###### Limitations:
- File names can only be 16 characters including `'\0'`
- No directories
- No permissions
